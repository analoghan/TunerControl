DI Injection Timing Calculator — Help

Computes optimal Start of Injection (SOI) timing for direct injection engines based on cam profile and engine parameters. Outputs values ready for the MoTeC M1 Fuel Timing Primary Main table.

How DI Timing Differs from Port Injection

In a port injection engine, fuel is sprayed onto the back of the intake valve and can sit there until the valve opens. The timing reference is End of Injection and values advance with RPM as pulse width grows.

In a direct injection engine, fuel is sprayed directly into the combustion chamber. The timing reference is Start of Injection and the window is much more constrained:

  • Injection must start after the exhaust valve closes — otherwise fuel escapes out the exhaust
  • Injection must finish before the ignition event — otherwise fuel sprays into the flame front
  • Injection into the compression stroke should be avoided — it can cause misfires, detonation, and piston damage
Critical: Injecting fuel during the compression stroke risks spraying directly into the flame front. This can cause misfires, knock/detonation, and in severe cases, piston or injector damage. Always verify your pulse width stays within the available intake-stroke window.

Input Parameters

Cam Profile

  • Intake Duration @ .050" — Total intake lobe duration measured at 0.050" lift (from cam card)
  • Exhaust Duration @ .050" — Total exhaust lobe duration at 0.050" lift
  • Intake Centerline (° ATDC) — The intake lobe centerline in degrees After Top Dead Center (overlap TDC). Found on the cam card or measured with a degree wheel.
  • Lobe Separation Angle (LSA) — The angle between intake and exhaust lobe centerlines. Fixed by the cam grind.
  • Cam Advance — How many degrees the cam is advanced from its neutral position. Positive = advanced (intake events happen earlier).
  • Overlap @ .050" — Degrees of valve overlap at 0.050" lift. With mild cams this may be 0-5°; aggressive cams can be 20-40°+.
Where to find this data: Your cam manufacturer's cam card will list duration, lift, LSA, and recommended centerline. If you've degree'd the cam, use your measured intake centerline. If not, use the manufacturer's recommended installed centerline.

Engine Parameters

  • Max RPM — The highest RPM you'll operate at. Used to generate the timing table breakpoints.
  • RPM Step — Spacing between RPM breakpoints in the output table (typically 500).
  • Safety Margin Before Ignition — Minimum degrees of crank rotation between end of injection and the spark event. Default 15° provides a safe buffer.
  • Max Spark Advance (° BTDC) — The most advanced ignition timing your tune will use. This determines the earliest possible ignition event in the cycle.

Understanding the Outputs

Valve Events

Shows all four valve events (EVO, EVC, IVO, IVC) converted to MoTeC's 720° reference scale:

0° = Power stroke TDC (firing)

180° = BDC (end of power stroke)

360° = TDC overlap (between exhaust and intake)

540° = BDC (end of intake stroke / start of compression)

720° = TDC compression (ignition point)

DI Injection Window

The safe zone for injection: starts after EVC (exhaust valve closed) and ends before the ignition event minus your safety margin. The tool reports:

  • Optimal SOI (° ATDC) — The recommended start point in degrees after TDC overlap
  • MoTeC Programmed Value — The number to enter in the Fuel Timing Primary Main table. Calculated as 360 - SOI_ATDC.
  • Available window — Total crank degrees available for injection

Maximum Injection Time vs RPM

Shows the maximum injector pulse width (in milliseconds) at each RPM before the injection event crosses into the compression stroke. This is the critical limit:

  • Green (>8ms) — Plenty of room, no concerns
  • Orange (4-8ms) — Getting tight, monitor actual pulse width
  • Red (<4ms) — Very limited window, likely need split injection at this RPM
If your logged pulse width exceeds the max PW at a given RPM: The injection is extending into the compression stroke. Options: (1) verify injector sizing is adequate, (2) increase fuel pressure to reduce pulse width, (3) use split injection (two shorter events instead of one long one).

SOI Timing Table

The recommended Start of Injection value for each RPM breakpoint. For DI engines, this is typically a flat value across all RPMs because the injection window is defined by crank angle (not time). Copy this directly into MoTeC's Fuel Timing Primary Main table.

Injection Duration Analysis

Shows what happens at a reference pulse width (6ms) across RPM. Validates that:

  • Injection starts after EVC (fuel won't escape through exhaust)
  • Injection doesn't extend into compression (avoid flame-front contact)
  • End of injection is clear of the ignition zone

MoTeC M1 Setup

  1. Navigate to Fuel Timing Primary in the M1 workbook
  2. Set Edge to Start of Injection
  3. Enter the calculated programmed value across all RPM breakpoints
  4. The MAP axis values can all be the same (DI timing doesn't typically vary with load)
  5. Verify with logged data that SOI and EOI fall within the expected window
Tuning tip: Once your VE table is dialed in and lambda is on target, try adjusting SOI ±10-20° and watch for lambda changes. If the engine goes richer without any VE change, you've found a more efficient injection point where more fuel is being utilized in combustion.

Cam Overlap Considerations

Engines with aggressive cams (high overlap) have a narrower DI injection window because the exhaust valve closes later. Key points:

  • Mild cams (0-10° overlap): Wide window, no concerns. SOI just after TDC overlap works well.
  • Moderate cams (10-30° overlap): Window narrows. Verify max PW table at high RPM.
  • Aggressive cams (30°+ overlap): May require retarding SOI significantly. Consider if DI-only fueling is viable or if supplemental port injection is needed at high RPM/load.

Split Injection

When a single injection event can't fit within the intake stroke at high RPM/load, split injection divides the fuel into two events:

  1. Primary event (intake stroke): ~70-80% of fuel mass, timed per this calculator
  2. Secondary event (compression stroke): ~20-30% of fuel mass, timed 30-60° BTDC compression for charge cooling and mixture preparation

Split injection is configured separately in MoTeC using the secondary injection timing tables. This calculator covers the primary event only.

Troubleshooting

Raw fuel smell from exhaust

SOI is too early — fuel is being injected while the exhaust valve is still open. Increase the programmed SOI value (retard the injection start) to move it further into the intake stroke.

Misfires at high RPM

EOI may be too close to the ignition event, or injection is extending deep into the compression stroke. Check the max PW table and verify logged pulse width is within limits.

Lambda reads lean at high RPM despite adequate injector sizing

Injection may be finishing too late — fuel doesn't have time to vaporize and mix before ignition. Try advancing SOI slightly (lower programmed value) to give more mixing time.
